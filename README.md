# COTANA BOT

COTANA is an advanced WhatsApp bot built with Baileys.

## Features
- AI Chat
- Media Downloader (YouTube, TikTok, Facebook, etc.)
- Group Management
- Sticker Creation
- And more!

## Setup

1. **Prerequisites**
   - Node.js (Latest)
   - MongoDB Atlas account

2. **Configuration**
   - Rename `.env.example` to `.env` (already done in this setup)
   - Add your `MONGODB_URI` from MongoDB Atlas
   - Add your `PHONE_NUMBER` (with country code)

3. **Installation**
   ```bash
   npm install
   ```

4. **Start**
   ```bash
   npm start
   ```

## Connect
Once the bot starts, it will generate a pairing code. Enter this code in your WhatsApp Linked Devices section.
